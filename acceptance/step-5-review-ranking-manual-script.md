# Step 5 评审与排名手工验收脚本

- 执行时间：2026-03-23
- 执行环境：本地开发环境 `bun run dev` + 认证态浏览器 live QA
- 执行赛事：`ai-2026`
- 执行结果：主闭环通过
- 实际覆盖：管理员分配评委、评委进入 `/judge`、评委提交评分、后台汇总榜单、公示开启与关闭、移除已评分评委后榜单同步收缩
- 待补边界：已通过 E2E 自动化覆盖（见 `e2e/specs/review-ranking.spec.ts`）

## 验收目标

验证 Step 5 第一版闭环是否成立：

`管理员分配评委 -> 评委进入评审中心 -> 对终稿作品打分 -> 后台看到汇总榜单 -> 前台按开关公示排名`

## 前置条件

1. 本地开发环境已启动：`bun run dev`
2. 数据库可写
3. 已有一个已发布赛事，并且至少有 1 份终稿作品
4. 已有两个可登录用户账号：
   - 管理员账号
   - 普通评委账号（需先至少成功登录过一次，数据库里已有 `User` 记录）

## 建议测试数据

- 赛事：任选一个已进入评审阶段的已发布赛事
- 评委邮箱：任选一个已存在 `User` 记录的登录邮箱
- 示例评分：
  - 创新性：9
  - 完成度：8
  - 落地价值：8.5
- 示例评语：`方案完整度高，建议进一步补充落地数据。`

## 验收步骤

### 场景 A：管理员分配评委

1. 管理员登录，进入 `/admin/events/[id]/judging`
2. 在“评委与公示”区块输入评委邮箱
3. 点击“分配评委”

**预期结果**

- 页面不报错
- 评委列表新增该用户
- 列表中能看到邮箱与已评分作品数

继续验证负向：

4. 重复输入同一邮箱再次分配

**预期结果**

- 返回明确提示：该用户已被分配为此赛事评委

### 场景 B：评委进入评审中心

1. 使用评委账号登录
2. 打开首页 Header，确认出现“评审中心”
3. 进入 `/judge`

**预期结果**

- 页面可正常打开
- 能看到被分配赛事卡片
- 卡片展示评审时间、作品总数、已评数量、待评数量

### 场景 C：评委评分

1. 进入 `/judge/events/[id]`
2. 在左侧选择一份终稿作品
3. 在右侧评分表填写各维度分数与评语
4. 点击“保存评分”

**预期结果**

- 页面返回“评分已保存”
- 左侧该作品状态变为“已保存评分”
- 重新打开该作品时，分数与评语会正确回填

继续验证负向：

5. 故意将某个分数填写为超出最高分的值，例如 `11`
6. 再次点击“保存评分”

**预期结果**

- 提交被阻止
- 对应字段显示“分数需在 0 到 X 之间”

### 场景 D：后台查看汇总

1. 切回管理员账号
2. 刷新 `/admin/events/[id]/judging`

**预期结果**

- “已评分作品”数量更新
- 榜单区域出现已评分作品
- 榜单显示排名、作品名、赛道、评分人数、平均总分

继续验证移除评委：

3. 在评委列表中移除刚刚完成评分的评委
4. 再次观察“已评分作品”数量与榜单区域

**预期结果**

- 该评委从分配列表中消失
- 该评委历史评分不再计入“已评分作品”数量
- 若该作品仅有这 1 位评委评分，对应榜单记录会同步消失
- 若作品还有其他评委评分，榜单会按剩余有效评分重新汇总

### 场景 E：前台公示排名

1. 在后台评审页点击“公开排名”
2. 打开 `/events/[slug]`

**预期结果**

- 前台出现“公开榜单”区块
- 榜单内容与后台汇总一致

继续验证：

3. 回后台点击“关闭排名公示”
4. 刷新前台赛事详情页

**预期结果**

- 榜单区块消失

### 场景 F：权限与窗口期边界

1. 使用未被分配的普通用户访问 `/judge/events/[id]`

**预期结果**

- 无法正常进入该赛事评审页

2. 将赛事时间调整到评审窗口外，再用已分配评委进入评分页

**预期结果**

- 页面提示当前不在评审时间窗口内
- 评分表按钮禁用

### 场景 G：已有评分后冻结评分维度

1. 保持场景 C 中至少一份作品已经成功保存评分
2. 切回管理员账号，进入 `/admin/events/[id]/edit`
3. 修改任一评分维度的名称、最高分、权重，或仅调整评分维度顺序
4. 点击“保存修改”

**预期结果**

- 表单提交被阻止
- 页面显示明确提示：已有评分后不可修改评分维度
- 赛事其他配置未被部分写入

继续验证：

5. 保持评分维度不变，仅修改赛事描述等普通字段
6. 再次点击“保存修改”

**预期结果**

- 保存成功
- 已有评分与榜单汇总不受影响
