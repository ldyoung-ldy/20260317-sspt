# Step 2 赛事管理验收清单

## 1. 后台赛事创建页 `/admin/events/new`

### 基础信息
- [ ] 页面可正常打开
- [ ] 可填写“赛事名称”
- [ ] 可填写“赛事描述”
- [ ] 未报错时可提交

### 时间窗口
- [ ] 可填写 8 个时间字段
- [ ] 合法时间顺序可提交
- [ ] 非法顺序会提示校验错误
- [ ] `registrationStart < registrationEnd`
- [ ] `registrationEnd <= submissionStart`
- [ ] `submissionStart < submissionEnd`
- [ ] `submissionEnd <= reviewStart`
- [ ] `reviewStart < reviewEnd`
- [ ] `reviewEnd <= endDate`

### 配置区块
- [ ] 可新增/删除赛道
- [ ] 可新增/删除赛题
- [ ] 可新增/删除奖项
- [ ] 可新增/删除评分维度
- [ ] 至少 1 个评分维度才能提交
- [ ] 评分维度名称重复时会报错
- [ ] 可新增/删除自定义报名字段
- [ ] 自定义字段类型切换正常
- [ ] `select` 类型未填选项时会报错

### 提交结果
- [ ] 提交成功后跳回 `/admin/events`
- [ ] 新赛事默认是“草稿”
- [ ] slug 自动生成成功

## 2. 后台赛事列表页 `/admin/events`

### 列表展示
- [ ] 页面可正常打开
- [ ] 能看到新创建的赛事
- [ ] 显示名称、slug、描述摘要
- [ ] 显示阶段 badge
- [ ] 显示发布状态 badge
- [ ] 显示赛道/奖项/评分维度数量

### 空状态
- [ ] 没有赛事时显示空状态提示
- [ ] 空状态下“立即创建”按钮可跳转

### 发布操作
- [ ] 草稿赛事可点击“发布”
- [ ] 已发布赛事可点击“取消发布”
- [ ] 发布后状态立即刷新
- [ ] 草稿时“预览”不可用
- [ ] 发布后“预览”可进入详情页

### 已发布赛事展示

- [ ] 只显示“已发布”赛事
- [ ] 草稿赛事不会出现在首页
- [ ] 赛事卡片显示名称
- [ ] 赛事卡片显示 slug
- [ ] 赛事卡片显示描述摘要
- [ ] 赛事卡片显示阶段
- [ ] 赛事卡片显示时间范围
- [ ] 赛事卡片显示赛道/奖项/评分维度数量
- [ ] 赛事卡片显示报名截止时间

### 空状态
- [ ] 没有已发布赛事时显示空状态
